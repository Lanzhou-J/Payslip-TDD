# Payslip

An OOP style basic payslip console app written in C#.
This project is to practise TDD.

---

## Start
#### Use Command Line:
1. cd into Payslip-TDD directory
    ```
    cd Payslip-TDD
    ```
2. start app
    ```
    dotnet run
    ```
#### Use JetBrains Rider:
1. Click "Run" button in Rider IDE (⌥F5). 

## Run Tests

#### Use Command Line:
1. ?
#### Use JetBrains Rider:
1. Select "Tests" -> "Run Unit Tests" (⌘U).
